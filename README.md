<h1>STIGS Vulnerability Remediations</h1>
  <p class="subtitle">Windows 11 STIG remediation scripts prepared for GitHub documentation.</p>

  <div class="section">
    <ul>
      <li><code>WN11-00-000155</code> — Disable PowerShell 2.0</li>
      <li><code>WN11-00-000175</code> — Disable Secondary Logon Service</li>
      <li><code>WN11-AC-000040</code> — The built-in Microsoft password complexity filter must be enabled</li>
      <li><code>WN11-AU-000505</code> — The Security event log size must be configured to 1024000 KB or greater</li>
      <li><code>WN11-CC-000038</code> — WDigest Authentication must be disabled</li>
      <li><code>WN11-CC-000050</code> — Configure Hardened UNC Paths</li>
      <li><code>WN11-CC-000206</code> — Windows Update must not obtain updates from other PCs on the internet</li>
      <li><code>WN11-CC-000280</code> — Remote Desktop Services must always prompt a client for passwords upon connection</li>
      <li><code>WN11-CC-000327</code> — Enable PowerShell Transcription Logging</li>
      <li><code>WN11-SO-000070</code> — The machine inactivity limit must be set to 15 minutes, locking the system with the screensaver</li>
    </ul>
  </div>
